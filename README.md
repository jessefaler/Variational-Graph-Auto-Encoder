## Variational Graph Auto-Encoder

This is an implementation of the Variational Graph Auto-Encoder described in the paper:  
https://arxiv.org/pdf/1611.07308

The model is trained on the Cora dataset.

## Setup

Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python run.py
```
